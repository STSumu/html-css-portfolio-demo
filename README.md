# Personal Portfolio Website

A simple portfolio website built using **HTML** and **CSS** during my early days of learning web development.

## Features

* Portfolio landing page
* About section
* Skills section
* Resume section
* Contact form
* Responsive layout

## Technologies Used

* HTML5
* CSS3
* Font Awesome
* Google Fonts

## Live Demo

View the project here:
https://stsumu.github.io/html-css-portfolio-demo/

## Running the Project

Clone the repository and open `index.html` in your browser.
